# Checkout SDK

`@arc-checkout/sdk` exposes invoice creation, retrieval, status, merchant payment listing, typed errors, and webhook verification.

```ts
import { ArcCheckout } from "@arc-checkout/sdk";

const checkout = new ArcCheckout({
  apiUrl: process.env.ARC_CHECKOUT_API_URL!,
  apiKey: process.env.ARC_CHECKOUT_API_KEY,
});

const invoice = await checkout.paymentIntents.create({
  merchantAddress: "0x...",
  orderId: "ORDER-1042",
  amount: "100.00",
  expiresInSeconds: 3600,
  refundAddress: "0x...",
  description: "Order #1042",
});
```

Idempotency keys are generated by default but production merchants should persist and reuse the same key when retrying one logical create request.
